\# File Access Monitor



\## О проекте



Это приложение для мониторинга и протоколирования событий доступа к системным ресурсам Windows. Разработано в рамках курсовой работы по дисциплине «Защита данных».



Программа позволяет отслеживать:

\- \*\*Файловые папки\*\* — чтение, запись, удаление файлов

\- \*\*Принтеры\*\* — отправка, завершение и ошибки печати

\- \*\*Разделы реестра\*\* — чтение и запись параметров



\## Возможности



\- Мониторинг файловых папок (чтение, запись, удаление)

\- Мониторинг принтеров (отправка на печать, завершение печати, ошибка печати)

\- Мониторинг разделов реестра (чтение, запись)

\- Фильтрация событий по типу доступа

\- Сохранение логов в текстовый файл

\- Графический пользовательский интерфейс на Windows Forms

\- Работа в реальном времени



\## Системные требования



| Требование | Минимальное |

|------------|-------------|

| \*\*ОС\*\* | Windows 10 / 11 |

| \*\*Права\*\* | Администратор (для мониторинга папок и реестра) |

| \*\*.NET\*\* | Framework 4.8 или выше |

| \*\*ОЗУ\*\* | 512 МБ |

| \*\*Место на диске\*\* | 50 МБ |



\## Установка и запуск



1\. Перейдите в раздел \*\*\[Releases](https://github.com/nastasyaR10/projects/tree/main/C%23-projects/FileAccessMonitoring/bin/Release)\*\*

2\. Скачайте файл `КР\_ЗД.exe`

3\. Запустите программу \*\*от имени администратора\*\* (можно пропустить этот пункт, если интересует только мониторинг принтера)
